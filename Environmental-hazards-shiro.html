Environmental hazards + VFX

Project: SHIRO

  • Category: Programming & VFX
  • Client: Tier 21
  • Project date: Q3-Q4, 2021
  • Project URL: https://store.steampowered.com/app/1589330/SHIRO/

Although the most noticeable in this task are the VFX my role in this task was to create environmental hazards that would be used by the designers to build levels mechanics. These were quite diverse in terms of game mechanics, and VFX. I started creating a base component that all environmental hazards would inherit, containing the basic methods, variables, and public API for external uses. The VFX part was made by me using the built-in particles system, shader graph, and an optimized camera setup for screen distortion effects in 2D.
