Personal project of an 2D metroidvania

Project information

  • Category: Game/prototype
  • Project date: Q1-Q2, 2020
  • Project URL: www.example.com

The first personal project I did, starting in Unity and C # only with prior knowledge of Python and Java. The idea of the game is a 2D Metroidvania with fast-paced combat and environment puzzle solving. The first challenge I had been through was the animation system that could handle tons of animations (made by myself in Blender and exported as sprite sheets), and the fluid combat system that supports tons of attacks and transitions. I also built a dynamic level loading system to support a seamless open world. I also had some experience with photoshop drawing scenes.
