Enemies A.I. for SHIRO project

Project information

  • Category: Programming
  • Client: Tier 21
  • Project date: Q3-Q4, 2021
  • Project URL: Steam page

The enemies in Shiro are diverse in terms of complexity and implementation, they were made using the tool I built before, with lots of new features that were implemented especially for this project. The whole system uses a state machine pattern with a Node-based editor that can make the behavior complex and easy to understand, they can be easily edited by anyone in the project. Also, the grounded enemies use a pathfinding solution for 2D platform games that makes them smarter and autonomous, together with a special system called "Vantage point system" that they can use to auto-positioning in the environment.
