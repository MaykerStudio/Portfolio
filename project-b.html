Personal project of an arcade android game

Project information

  • Category: Prototyping
  • Project date: Q1-Q2, 2020
  • Project URL: Download android demo

This project started as an experience to show off how fast I could create a prototype game for android. The main goal of the game is to protect the blue shield for 5 minutes. Enemies can drop weapons that could help kill them easily. Some enemies can only be killed using filter glasses to target them. The core gameplay of the game is ready, the plan was to add more enemies, new levels, and even bosses for the game to feel more diverse.
